## Keybindings

| Function                                    | Keybind            |
| ------------------------------------------- | ------------------ |
| Goto Line                                   | `Ctrl + G`         |
| Goto Symbol                                 | `Ctrl + Shift + O` |
| Select all occurrences of current selection | `Ctrl + Shift + l` |
| Show all Symbols                            | `Ctrl + T`         |
